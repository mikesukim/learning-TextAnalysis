Assignment 9

  1. In this assignment, you will be writing a Python program which reads a text file containing a story and another with a comma separated list of words to be ignored (i.e. a "skip words" file). It will read the contents of both and process them to remove the words in the skip words file from the story text, and then develop a listing of the 5 most common pairs of consecutive words in the remaining text of the story.
    Note that your program should work using Python 3 (e.g. "python3" from the command line of the virtual box).


  2. Your program must take two command line arguments, in the following order:
    1. the name of the text file containing the story
    2. the name of the text file containing the words to be skipped (which will be formatted as a comma separated list)
    Examples of each are included with the assignment.
    If there are not enough arguments provided via the command line, it should print a descriptive error message stating that. Hint: use "import sys", then you can access the command line argument array via "sys.argv". (The first entry will be the name of the Python script being executed.)


  3. Open each file, read in its contents, then close it.
    Hint: Using the command "with open(storyFileName, 'r') as storyFile:" will cause the file whose name is stored in the variable storyFileName to be opened and accessible via the variable storyFile. At the end of the "with" block, the file will automatically be closed for you.


  4. Lowercase the story text to make it easier to work with.
    Hint: There is a string method which can do this.


  5. Remove each character from the following array from the story text:
    erasures = ['\n','\t','.','?','!',',',';',':','\'','\"']
    Hint: You may find the string methods `split' and `replace' useful. If you don't know how to use them, search Google for documentation.


  6. Make a list containing the words in the story, in the order they appear. (Note: this will essentially turn the story text into a list of words, using the spaces between them to separate them.)


  7. Remove each word contained in skip words file from the story text.


  8. Make a dictionary where the keys are strings and the values are integers. For each pair of consecutive words in the (remaining) story text, there should be a key containing those words (in that order) separated by a space, and the value associated with that key should be the number of times that that those words appear consecutively in that order within the story text. (You need to figure out how to do this.)


  9. If you import the "operator" package (e.g. "import operator"), the following line can be used to sort a dictionary by its values, from smallest to largest (where the dictionary being sorted is named "pairCount"):
    "sorted(pairCount.items(), key=operator.itemgetter(1))"


  10. Output the top five most frequently occurring word pairs, along with their counts. I've provided an example story in "emperor3.txt" and a file of words to be skipped, "skip-words.txt". (That story and others can be found here: http://www.textfiles.com/stories/.) The output from my solution running with those input files follows.
